# DIY Arduino CNC Laser Machine

## Project Overview
This is a DIY CNC laser engraving machine built using Arduino Uno and GRBL firmware. It can engrave text and simple designs on wood, cardboard, and other suitable materials.

## Components Used
- Arduino Uno
- CNC Shield
- A4988 Stepper Drivers
- NEMA 17 Stepper Motors
- Laser Module
- Power Supply
- GT2 Belt and Pulley
- Frame

## Software Used
- Arduino IDE
- LaserGRBL
- Inkscape
- GRBL Firmware

## Build Steps

### Step 1
Collect all the required components.

### Step 2
Assemble the machine frame.

### Step 3
Mount the X-axis and Y-axis stepper motors.

### Step 4
Install the laser module.

### Step 5
Connect the CNC Shield and A4988 drivers to the Arduino Uno.

### Step 6
Upload the GRBL firmware using Arduino IDE.

### Step 7
Test motor movement.

### Step 8
Generate G-code using Inkscape.

### Step 9
Open LaserGRBL and connect the machine.

### Step 10
Run the G-code and perform your first engraving.
